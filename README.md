## Get started

### Requirements

- Python version: 3.10.9
- OS environment: alpine linux
- Docker environment: in progress
- Database engine: MongoDB 4.0.8

### How to use

Run container

```
  docker-compose up -d
```

Close docker-compose

```
  docker-compose down -v
```
